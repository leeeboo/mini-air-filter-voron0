# mini-air-filter-voron0

A mini air filter use carbon and one 5015 blower.
For small 3D Printer such as Voron0.1.

### BOM

* 6x3mm Magnets X 8
* M3x8 Screw X 4
* M4x22 Screw X 2
* M4 Nut X 2
* 5015 Blower X 1
* Active carbon
